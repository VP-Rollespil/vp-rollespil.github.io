A wiki generator for Valbyparkens Rollespil

Syntax is based on Wikitext with support for:

- Headers
- Lists
- External and internal links
- Code
- Bold (using ''')
- Italic (using '')
- Underline (using \_\_)
- Strikethrough (using ~~)
- horizontal rule (using ----)
